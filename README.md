# Loan Approval Prediction using Bayesian Network

## 📌 Overview

This project uses a Bayesian Network to predict loan approval based on applicant financial attributes such as income, credit score, loan amount, and assets.

## ⚙️ Features

* Probabilistic prediction (Approved / Rejected)
* Manual review for uncertain cases
* Web-based UI using Flask
* Model evaluation with accuracy, precision, recall, and F1 score

## 🧠 Model

* Bayesian Network (pgmpy)
* Bayesian Estimator
* Variable Elimination for inference

## 📊 Tech Stack

* Python
* pgmpy
* Flask
* scikit-learn

## 🚀 How to Run

```bash
pip install -r requirements.txt
python -m src.app
```
